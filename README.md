# ts-starter

A typescript starter

## Contents

- [Features](#features)
- [Usage](#usage)
- [License](#license)

## Features

- No linter, just `prettier`
- Compiles with `tsup`
- Runs typescript with `esrun`
- Tests with `vitest`

## Usage

```sh
npx degit hankaguila/ts-starter
```
